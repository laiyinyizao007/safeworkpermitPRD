# 7. 非功能性需求

## 7.1 性能要求

### 7.1.1 响应时间

**Web端响应时间**:

| 操作类型 | 目标响应时间 | 最大可接受时间 | 备注 |
|---------|-------------|---------------|------|
| 页面加载 | ≤ 2秒 | ≤ 5秒 | 首屏加载时间 |
| 列表查询 | ≤ 1秒 | ≤ 3秒 | 单页50条数据 |
| 表单提交 | ≤ 1秒 | ≤ 3秒 | 不含文件上传 |
| 文件上传 | ≤ 5秒 | ≤ 10秒 | 单文件≤10MB |
| 报表生成 | ≤ 5秒 | ≤ 15秒 | 数据量≤10万条 |
| 地图加载 | ≤ 3秒 | ≤ 8秒 | 含GIS数据 |

**移动端响应时间**:

| 操作类型 | 目标响应时间 | 最大可接受时间 | 备注 |
|---------|-------------|---------------|------|
| 页面加载 | ≤ 3秒 | ≤ 6秒 | 4G网络环境 |
| 列表查询 | ≤ 2秒 | ≤ 4秒 | 单页30条数据 |
| 表单提交 | ≤ 2秒 | ≤ 4秒 | 不含文件上传 |
| 拍照上传 | ≤ 3秒 | ≤ 8秒 | 单张照片≤5MB |
| 扫码识别 | ≤ 1秒 | ≤ 2秒 | 二维码/条形码 |
| 人脸识别 | ≤ 2秒 | ≤ 5秒 | 本地识别 |

**IoT设备响应时间**:

| 操作类型 | 目标响应时间 | 最大可接受时间 | 备注 |
|---------|-------------|---------------|------|
| 数据采集 | ≤ 1秒 | ≤ 3秒 | 传感器数据 |
| 数据上报 | ≤ 2秒 | ≤ 5秒 | 含网络传输 |
| 指令下发 | ≤ 1秒 | ≤ 3秒 | 设备控制指令 |
| 告警推送 | ≤ 3秒 | ≤ 10秒 | 含多渠道推送 |

### 7.1.2 并发能力

**用户并发**:
- 支持1000个用户同时在线
- 支持100个用户同时提交作业票
- 支持50个用户同时审批作业票
- 支持200个用户同时查看作业票

**设备并发**:
- 支持500个IoT设备同时在线
- 支持100个IoT设备同时上报数据
- 支持50个视频流同时播放

**数据处理能力**:
- 支持每秒处理1000条IoT数据
- 支持每秒处理100个告警事件
- 支持每秒处理50个审批请求

### 7.1.3 数据库性能

**查询性能**:
- 单表查询响应时间 ≤ 100ms (数据量≤100万条)
- 多表关联查询响应时间 ≤ 500ms (关联表≤5个)
- 全文检索响应时间 ≤ 1秒 (数据量≤1000万条)

**写入性能**:
- 单条数据写入响应时间 ≤ 50ms
- 批量数据写入响应时间 ≤ 5秒 (批量≤1000条)
- 事务提交响应时间 ≤ 100ms

**数据库连接池**:
- 最小连接数: 10
- 最大连接数: 100
- 连接超时时间: 30秒

### 7.1.4 缓存策略

**缓存类型**:
- 页面缓存(静态资源)
- 数据缓存(热点数据)
- 会话缓存(用户状态)
- 查询缓存(查询结果)

**缓存有效期**:
- 静态资源: 7天
- 热点数据: 1小时
- 用户状态: 30分钟
- 查询结果: 5分钟

**缓存更新策略**:
- 数据变更时主动失效
- 定时刷新(每5分钟)
- LRU淘汰策略

## 7.2 可靠性要求

### 7.2.1 系统可用性

**可用性指标**:
- 系统年可用性 ≥ 99.5% (允许年停机时间 ≤ 43.8小时)
- 核心功能可用性 ≥ 99.9% (允许年停机时间 ≤ 8.76小时)
- 计划内维护窗口: 每月1次,每次≤4小时

**故障恢复时间**:
- 故障检测时间 ≤ 5分钟
- 故障定位时间 ≤ 30分钟
- 故障修复时间 ≤ 2小时
- 数据恢复时间 ≤ 1小时

### 7.2.2 数据可靠性

**数据备份**:
- 全量备份: 每天1次(凌晨2:00)
- 增量备份: 每小时1次
- 备份保留期: 30天
- 备份存储: 异地备份(至少2个地点)

**数据恢复**:
- 支持任意时间点恢复(RPO ≤ 1小时)
- 数据恢复时间 ≤ 4小时(RTO ≤ 4小时)
- 数据恢复成功率 ≥ 99%

**数据一致性**:
- 强一致性: 作业票数据、审批记录
- 最终一致性: 统计数据、日志数据
- 事务隔离级别: READ COMMITTED

### 7.2.3 容错能力

**单点故障容错**:
- 应用服务器: 至少2台(主备或负载均衡)
- 数据库服务器: 主从复制(1主2从)
- 缓存服务器: 集群模式(至少3节点)
- 消息队列: 集群模式(至少3节点)

**故障自动切换**:
- 应用服务器故障自动切换时间 ≤ 30秒
- 数据库主从切换时间 ≤ 1分钟
- 缓存节点故障自动剔除时间 ≤ 10秒

**降级策略**:
- 非核心功能降级(如统计报表)
- 限流保护(超过并发阈值时排队)
- 熔断保护(依赖服务故障时熔断)

### 7.2.4 异常处理

**异常分类**:
- 系统异常(服务器故障、网络故障)
- 业务异常(数据校验失败、权限不足)
- 第三方异常(IoT设备离线、CA认证失败)

**异常处理策略**:
- 系统异常: 记录日志、告警通知、自动重试(最多3次)
- 业务异常: 友好提示、记录日志
- 第三方异常: 记录日志、告警通知、降级处理

**异常恢复**:
- 自动重试机制(指数退避)
- 人工干预机制(工单系统)
- 补偿机制(数据补偿、流程补偿)

## 7.3 安全性要求

### 7.3.1 身份认证

**认证方式**:
- 用户名密码认证(支持密码强度校验)
- 短信验证码认证(登录、重置密码)
- 人脸识别认证(移动端签到)
- CA数字证书认证(电子签名)
- 单点登录(SSO,可选)

**密码策略**:
- 密码长度 ≥ 8位
- 密码必须包含大小写字母、数字、特殊字符
- 密码有效期: 90天
- 密码历史: 不能与最近5次密码相同
- 密码错误锁定: 连续5次错误锁定30分钟

**会话管理**:
- 会话超时时间: 30分钟(无操作自动退出)
- 单用户单会话(同一用户只能在一个设备登录)
- 会话令牌: JWT(有效期30分钟,支持刷新)

### 7.3.2 权限控制

**权限模型**:
- RBAC(基于角色的访问控制)
- 支持角色继承
- 支持数据权限(部门、区域)
- 支持功能权限(菜单、按钮)

**权限粒度**:
- 菜单级权限(是否可见)
- 按钮级权限(是否可操作)
- 数据级权限(是否可查看/编辑)
- 字段级权限(是否可查看/编辑)

**权限审计**:
- 记录所有权限变更
- 记录所有敏感操作(审批、删除等)
- 审计日志保留期 ≥ 1年

### 7.3.3 数据加密

**传输加密**:
- HTTPS(TLS 1.2+)
- WebSocket加密(WSS)
- IoT设备通信加密(MQTT over TLS)

**存储加密**:
- 敏感字段加密(密码、身份证号、手机号)
- 加密算法: AES-256
- 密钥管理: 密钥分离存储、定期轮换

**文件加密**:
- 上传文件加密存储
- 下载文件动态解密
- 文件访问权限控制

### 7.3.4 安全审计

**审计内容**:
- 用户登录/登出
- 权限变更
- 数据增删改
- 敏感操作(审批、删除、导出)
- 系统配置变更

**审计日志格式**:
- 操作时间
- 操作人
- 操作IP
- 操作类型
- 操作对象
- 操作结果
- 操作详情

**审计日志存储**:
- 日志保留期 ≥ 1年
- 日志不可篡改(区块链存证,可选)
- 日志定期归档

### 7.3.5 防攻击措施

**SQL注入防护**:
- 使用参数化查询
- 输入数据校验
- 敏感字符过滤

**XSS防护**:
- 输出数据转义
- CSP(内容安全策略)
- HttpOnly Cookie

**CSRF防护**:
- CSRF Token
- SameSite Cookie
- Referer校验

**DDoS防护**:
- 限流(IP限流、用户限流)
- 黑名单(IP黑名单、用户黑名单)
- CDN加速

**文件上传防护**:
- 文件类型校验(白名单)
- 文件大小限制
- 文件内容扫描(病毒扫描)
- 文件存储隔离

## 7.4 可维护性要求

### 7.4.1 日志管理

**日志分类**:
- 应用日志(业务日志、错误日志)
- 访问日志(HTTP访问日志)
- 审计日志(安全审计日志)
- 性能日志(慢查询日志、接口耗时日志)

**日志级别**:
- DEBUG: 调试信息
- INFO: 一般信息
- WARN: 警告信息
- ERROR: 错误信息
- FATAL: 致命错误

**日志格式**:
```json
{
  "timestamp": "2026-03-09T10:30:00.000Z",
  "level": "INFO",
  "logger": "com.example.service.WorkPermitService",
  "thread": "http-nio-8080-exec-1",
  "message": "作业票申请成功",
  "context": {
    "userId": "user123",
    "permitId": "WP20260309001",
    "permitType": "动火作业"
  }
}
```

**日志存储**:
- 本地文件存储(按天滚动)
- 集中式日志存储(ELK/EFK)
- 日志保留期: 30天(本地)、1年(集中式)

### 7.4.2 监控告警

**监控指标**:
- 系统指标(CPU、内存、磁盘、网络)
- 应用指标(QPS、响应时间、错误率)
- 业务指标(作业票数量、审批通过率、告警数量)
- 依赖指标(数据库连接数、缓存命中率、消息队列堆积)

**告警规则**:
- CPU使用率 > 80% 持续5分钟
- 内存使用率 > 85% 持续5分钟
- 磁盘使用率 > 90%
- 接口响应时间 > 3秒 持续1分钟
- 接口错误率 > 5% 持续1分钟
- 数据库连接数 > 80
- 消息队列堆积 > 1000条

**告警方式**:
- 短信告警(紧急告警)
- 邮件告警(一般告警)
- 企业微信/钉钉告警(实时告警)
- 电话告警(严重告警)

### 7.4.3 配置管理

**配置分类**:
- 系统配置(数据库连接、缓存配置)
- 业务配置(审批流程、时效规则)
- 功能开关(灰度发布、降级开关)

**配置中心**:
- 集中式配置管理(Nacos/Apollo)
- 配置热更新(无需重启)
- 配置版本管理
- 配置回滚

**配置变更**:
- 配置变更需审批
- 配置变更需测试
- 配置变更需通知
- 配置变更需记录

### 7.4.4 版本管理

**版本号规则**:
- 格式: 主版本号.次版本号.修订号 (如 1.2.3)
- 主版本号: 重大功能变更或架构调整
- 次版本号: 新增功能或功能优化
- 修订号: Bug修复或小优化

**版本发布**:
- 发布计划(提前1周通知)
- 发布窗口(每周五晚上或周末)
- 发布验证(冒烟测试)
- 发布回滚(发现问题立即回滚)

**版本兼容性**:
- 向后兼容(新版本兼容旧版本数据)
- API版本化(支持多版本API并存)
- 数据库版本化(Flyway/Liquibase)
