## 2. 组件依赖关系图

```mermaid
flowchart TD
    subgraph 核心组件
        FR[FormRenderer<br/>动态表单渲染器]
        AB[ActionBlock<br/>动作块]
    end

    subgraph 业务组件
        SIG[Signature<br/>电子签名]
        PU[PhotoUpload<br/>照片上传]
        GD[GasDetection<br/>气体检测]
        ES[EmergencyStop<br/>紧急叫停]
        ST[StatusTag<br/>状态标签]
        TL[Timeline<br/>审批时间轴]
        GF[GeoFence<br/>地理围栏]
        RS[RoleSwitcher<br/>角色切换]
    end

    subgraph 复合组件
        MP[MonitorPanel<br/>监护面板]
        SC[SafetyChecklist<br/>安全措施清单]
        RSM[RiskSummary<br/>风险摘要卡片]
    end

    FR -->|渲染字段时调用| SIG
    FR -->|渲染字段时调用| PU
    FR -->|渲染字段时调用| GD
    FR -->|渲染字段时调用| SC
    FR -->|渲染字段时调用| GF

    MP -->|内嵌| GD
    MP -->|内嵌| ES
    MP -->|内嵌| PU

    AB -->|读取| ST

    subgraph AI组件
        AIRS[AIReviewSidebar<br/>AI审查侧边栏]
        ARP[AuditResultPanel<br/>合规审计结果]
        ECP[ExpertConsensusPanel<br/>多专家共识]
        SVB[SpatialVerifyBadge<br/>时空验证徽章]
    end

    AIRS -->|locate-field| FR
    AIRS -->|auto-fill| FR
    AIRS -->|KG推荐采纳| SC
    ARP -.->|历史审计数据| AIRS
    ECP -->|嵌入| RSM
    SVB -->|验证状态| AB
    SVB -->|验证状态| SIG
    MP -->|在场监测| SVB
    GF -->|围栏数据| SVB
```

## 3. 组件与页面映射

| 页面 | 使用的组件 |
|------|-----------|
| 首页-待办中心 | StatusTag, ActionBlock, RoleSwitcher |
| 新建作业票 | FormRenderer, PhotoUpload, SafetyChecklist, **AIReviewSidebar** |
| 作业票详情页 | FormRenderer, ActionBlock, StatusTag, Timeline, Signature, MonitorPanel, GeoFence, EmergencyStop, RiskSummary, SafetyChecklist, GasDetection, PhotoUpload, **AuditResultPanel**, **ExpertConsensusPanel**, **SpatialVerifyBadge** |
| 电子签名页 | Signature, **SpatialVerifyBadge** |
| 管理后台 | —（使用 Element Plus 原生组件） |
