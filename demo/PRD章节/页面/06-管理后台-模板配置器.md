# 06 - 管理后台 - 模板配置器

> **路由**: `/admin/template` | **可见角色**: 仅系统管理员 | **关联组件**: FormRenderer（预览模式）
> **关联PRD**: [03-统一界面设计.md](../03-统一界面设计.md) §7.2 / 参考文档 §7.2
> **AI 集成**: Auditor Agent（AI 合规审查约束卡点）

---

## 1. 页面定位

可视化表单模板设计器，管理员通过拖拽组件配置作业票表单。三栏布局：左侧组件面板 + 中间拖拽画布 + 右侧属性面板。

## 2. 功能模块

| 模块 | 说明 |
|------|------|
| 组件面板（左栏） | 4大类可拖拽组件 |
| 拖拽画布（中栏） | 所见即所得的表单预览 |
| 属性面板（右栏） | 选中组件的属性配置 |
| 底部工具栏 | 组件数统计、版本号、预览/保存/发布按钮 |

## 4. 组件分类（4大类）

### 4.1 基础输入组件

| 组件 | 配置项 |
|------|--------|
| 单行文本 | 最小/最大长度、正则校验、占位符 |
| 多行文本 | 最大长度、行数 |
| 数字输入 | 最小/最大值、单位、步长 |
| 单选/多选 | 选项列表、默认值 |
| 下拉选择 | 选项列表、是否可搜索 |
| 日期/时间 | 格式、范围限制 |

### 4.2 业务逻辑组件

| 组件 | 配置项 |
|------|--------|
| 人员选择器 | 角色过滤、是否多选、联动组织架构 |
| 证书校验 | 证书类型、有效期校验规则 |
| 电子签名 | 签名模式（手写/CA）、是否必填 |
| 图片上传 | 最少张数、是否强制拍照、水印配置 |
| 地理位置 | 地图选点、GPS 定位 |

### 4.3 约束卡点组件（创新）

| 组件 | 配置项 | 执行逻辑 |
|------|--------|---------|
| 拍照卡点 | 最少张数、强制实时拍摄、AI 识别对象、水印 | 不满足条件时阻断提交 |
| 电子围栏 | 中心坐标、半径(米)、偏移容忍度 | 不在围栏内时表单置灰 |
| 人工确认清单 | 检查项列表、强制按序点击 | 前一项未勾选后一项无法操作 |
| 🤖 AI 合规审查 | 审查时机（提交前/保存时）、阻断等级（CRITICAL+HIGH 阻断）、审查智能体选择 | 调用 Auditor Agent 审查，不通过时阻断提交并展示 AIReviewSidebar |

### 4.4 容器与布局组件

| 组件 | 用途 |
|------|------|
| 卡片/分组 | 信息分块展示 |
| 动态明细表 | 一对多关系（如多个作业人员） |

## 5. 配置工作流

| 步骤 | 操作 | 说明 |
|------|------|------|
| Step 1 | 选择模板 | 从8大作业类型标准模板中选择 |
| Step 2 | 拖拽组件 | 从组件库拖入画布 |
| Step 3 | 配置属性 | 右侧面板设置字段属性和校验规则 |
| Step 4 | 设置联动 | 配置字段间的显示/隐藏联动逻辑 |
| Step 5 | 配置状态机 | 为每个状态配置可写字段、必填字段、约束卡点 |
| Step 6 | 预览发布 | 手机端/PC端预览，确认后发布 |

## 6. 页面导航

| 来源 | 操作 | 目标 |
|------|------|------|
| 系统概览 | 点击 [模板管理] | → 本页面（模板列表） |
| 模板列表 | 点击模板 | → 本页面（编辑模式） |
| 本页面 | 点击 [预览] | → 弹窗预览（手机端/PC端切换） |

## 7. Mock 数据接口

```typescript
// 获取模板列表
GET /api/admin/templates
Response: Template[]

// 获取模板详情
GET /api/admin/templates/:id
Response: { schema: FieldConfig[], layout: LayoutConfig, constraints: ConstraintConfig }

// 保存模板（Demo 仅前端状态）
PUT /api/admin/templates/:id
```

## 8. 交互适配

仅 PC 端。最小宽度 1280px，三栏比例约 200px : auto : 300px。
