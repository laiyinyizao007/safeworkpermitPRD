# 08 - 管理后台 - 权限矩阵编辑器

> **路由**: `/admin/permission` | **可见角色**: 仅系统管理员 | **关联组件**: —
> **关联PRD**: [03-统一界面设计.md](../03-统一界面设计.md) §7.4 / [04-角色权限视图差异.md](../04-角色权限视图差异.md)

## 1. 页面定位

三维权限配置界面：按「状态 × 角色 × 字段」配置字段级权限。管理员点击单元格循环切换权限类型。

## 2. 功能模块

| 模块 | 说明 |
|------|------|
| 状态选择器 | 顶部 Tab 切换当前编辑的状态 |
| 权限矩阵表格 | 行=字段，列=角色，单元格=权限类型 |
| 图例说明 | 权限类型图标说明 |
| 操作栏 | 保存/重置/导出按钮 |

## 4. 角色差异

本页面仅管理员可见，无角色差异。

## 5. 状态驱动 UI 变化

通过顶部状态 Tab 切换，表格内容随之变化，展示对应状态下的权限配置。

## 6. 页面导航

| 来源 | 操作 | 目标 |
|------|------|------|
| 系统概览 | 点击 [权限管理] | → 本页面 |

## 7. Mock 数据接口

```typescript
// 获取权限矩阵
GET /api/admin/permissions/:workType
Response: Record<PermitStatus, Record<string, Record<string, FieldPermission>>>

// 保存权限矩阵（Demo 仅前端状态）
PUT /api/admin/permissions/:workType
```

## 8. 交互适配

仅 PC 端。表格支持横向滚动（角色列较多时）。单元格点击循环顺序：✏️ → 👁️ → 🚫 → 📤 → ✅ → ✍️ → ✏️

## 9. Demo 简化说明

- 展示编辑交互，点击单元格可切换权限图标
- 不实现真实保存到后端，刷新后恢复默认
- 预置动火作业的完整权限矩阵数据
