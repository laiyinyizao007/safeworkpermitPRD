# 03 - 组件开发

## 3.1 NodeGraph 组件使用

### 3.1.1 基础用法

```vue
<template>
  <NodeGraph
    :nodes="workflowNodes"
    :edges="workflowEdges"
    :currentNode="currentNodeId"
    :width="1800"
    :height="300"
  />
</template>

<script setup>
import NodeGraph from '@/components/NodeGraph.vue';
import nodeLibrary from '@/config/nodeLibrary.json';

const workflowNodes = nodeLibrary.nodeGraph.hot_work_workflow.nodes;
const workflowEdges = nodeLibrary.nodeGraph.hot_work_workflow.edges;
const currentNodeId = ref('n1');
</script>
```

## 3.2 NodeExecutionMonitor 组件使用

### 3.2.1 监控节点执行

```vue
<template>
  <NodeExecutionMonitor :executor="nodeExecutor" />
</template>

<script setup>
import NodeExecutionMonitor from '@/components/NodeExecutionMonitor.vue';

const nodeExecutor = inject('nodeExecutor');
</script>
```

## 3.3 集成到主界面

### 3.3.1 布局结构

```vue
<template>
  <div class="work-permit-page">
    <!-- 节点流程图 -->
    <NodeGraph :nodes="nodes" :edges="edges" :currentNode="current" />

    <!-- 表单区域 -->
    <FormRenderer :schema="formSchema" v-model="formData" />

    <!-- 执行监控 -->
    <NodeExecutionMonitor :executor="executor" />
  </div>
</template>
```

## 3.4 响应式适配

移动端隐藏节点图，仅显示当前节点名称：

```vue
<div v-if="isMobile" class="current-node-badge">
  当前节点：{{ getCurrentNodeLabel() }}
</div>
<NodeGraph v-else :nodes="nodes" :edges="edges" />
```
