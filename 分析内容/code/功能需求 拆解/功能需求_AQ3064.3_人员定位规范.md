# AQ 3064.3—2025 人员定位规范 — 逐条需求模块解析

> 来源：AQ 3064.3—2025《"工业互联网+危化安全生产"建设规范 第3部分：人员定位》
> 生成日期：2026-03-10

---

## 一、总体要求（第4章）→ 基础架构模块

| 条款 | 原文要点 | 需求模块 |
|------|---------|---------|
| 4.1 | 符合AQ3064.1总体要求、总体架构、安全防护 | **系统架构合规模块** — 需对接总则定义的总体架构和安全防护要求 |
| 4.2 | 室内外定位硬件 + 电子地图可视化 + 历史轨迹 + 违规异常报警 | **定位基础设施模块**（硬件管理）、**电子地图模块**、**轨迹记录模块**、**报警引擎模块** |
| 4.3 | 技术路线需考虑遮挡、空间复杂度、精度需求 | **定位技术适配模块** — 支持多种定位技术（UWB/蓝牙/GPS等）按场景选型 |
| 4.4 | 数据存储≥1年，支持上传至园区 | **数据存储模块**（≥1年保留策略）、**数据上报模块**（企业→园区） |
| 4.5 | 软硬件可扩展性 | **系统扩展性设计** — 架构层面要求，插件化/模块化 |
| 4.6 | 覆盖生产区所有人员，进入必须携带终端 | **人员-终端绑定管理模块**、**进出管控模块** |

## 二、企业建设要求 — 5.1 信息管理 → 人员与设备管理模块

| 条款 | 原文要点 | 需求模块 |
|------|---------|---------|
| 5.1.1 | 人员信息管理（姓名/单位/岗位/资格/身份：内部员工、承包商、外来人员） | **人员信息管理模块** — CRUD + 三类身份分类 |
| 5.1.2 | 共享并同步更新其他系统的人员信息 | **人员数据同步接口** — 与HR系统、门禁系统等对接 |
| 5.1.3 | 终端管理（列表/电量/在线离线/绑定人/当前末次位置/加解绑）+ 相关设备管理 | **定位终端管理模块** — 设备台账、状态监控、绑定操作 |
| 5.1.4 | 历史轨迹查询（单人/多人/跨天/室内外连续自动切换） | **历史轨迹查询模块** — 多维度查询 + 室内外轨迹无缝拼接 |
| 5.1.5 | 电子围栏绘制 + 分区分类授权 + 三维立体围栏 | **电子围栏管理模块** — 围栏绘制、权限配置、3D围栏支持 |

## 三、5.2 预警报警管理 → 报警引擎模块（核心）

| 条款 | 原文要点 | 需求模块 |
|------|---------|---------|
| 5.2.1 | 报警信息含时间/人员/地点，实时听觉视觉提示，记录报警时刻定位，查看历史报警 | **报警信息记录与展示模块** — 实时推送 + 声光提示 + 历史查询 |
| 5.2.2 | 报警确认、闭环处置、设置处理流程和负责人 | **报警闭环处置工作流模块** — 确认→分派→处理→关闭 |
| 5.2.3 | 人员聚集预警算法（15m半径，持续>1min） | **人员聚集预警算法模块** |
| 5.2.3a | 3人黄色/4-6人橙色/6人以上红色，地图显示聚集位置范围数量人员清单，自动推送 | **聚集分级预警模块** — 三级颜色 + 地图标注 + 自动推送 |
| 5.2.3b | 聚集5s内自动警示分散；周边可燃/有毒气体报警时警示撤离 | **聚集联动警示模块** — 5s响应 + 气体报警联动 |
| 5.2.3c | 橙色/红色预警人工确认疏散并消警，保存记录 | **预警消警确认模块** |
| 5.2.3d | 确需聚集作业时，支持研判风险、完善措施、持续预警 | **聚集作业豁免管理模块** — 风险研判 + 持续监控 |
| 5.2.4a | 越界报警：5s内报警，30s不消警形成事件记录 | **越界报警模块** — 5s触发 + 30s升级 |
| 5.2.4b | 静止报警：危险区域超时不移动 | **静止报警模块** — 可配置区域和时间阈值 |
| 5.2.4c | 超员报警：区域人数超限 | **超员报警模块** — 可配置区域和人数阈值 |
| 5.2.4d | 缺员报警：区域人数不足或特定人员缺失 | **缺员报警模块** — 人数/特定人员双模式 |
| 5.2.4e | 滞留报警：区域滞留超时 | **滞留报警模块** — 可配置区域和时长 |
| 5.2.5 | 报警静默：设置静默类型、区域、有效时间 | **报警静默管理模块** — 停车检修等场景 |
| 5.2.6 | 终端发出报警 + 一键SOS求救 | **一键报警/SOS模块** — 终端侧触发 + 服务端接收推送 |

## 八、技术要求（第7章）→ 性能指标模块

| 条款 | 原文要点 | 需求模块/指标 |
|------|---------|---------|
| 7.1.1 | 蓝/黄区域：静态≤±3m，动态≤±5m；橙/红区域更高精度 | **定位精度管理** — 按风险等级分区配置精度要求 |
| 7.1.2 | 每年≥1次校准和准确度测试 | **定位校准管理模块** — 年度校准计划 + 测试记录 |
| 7.2.1 | 定位数据上传频率≥30次/min | **数据采集频率控制** — ≥2s/次 |
| 7.2.2 | 一键报警5s内推送 | **SOS推送性能** — 5s SLA |
| 7.2.3 | 页面响应≤1s，位置更新≤5s，8h轨迹查询≤3s | **前端性能指标** — 响应时间SLA |
| 7.2.4 | 室内外切换延迟≤3s | **室内外切换性能** — 3s SLA |
| 7.3.1 | 报警期间周期上报状态，频率≥12次/min | **报警状态上报频率** — ≥5s/次 |
| 7.3.2 | 漏报率<10%，误报率<15% | **报警准确性指标** — 漏报/误报率监控 |
| 7.3.3 | 每年≥1次漏报率误报率测试 | **报警质量年度测试模块** |
| 7.4 | 爆炸危险场所设备符合GB3836和GB50058 | **防爆合规管理** — 设备防爆等级记录 |

## 需求模块汇总（共约30个功能模块）

### 1. 人员管理域
- 人员信息管理（三类身份分类）
- 人员-终端绑定管理
- 人员数据同步接口

### 2. 设备管理域
- 定位终端管理（台账/状态/绑定）
- 定位技术适配（多技术路线）
- 防爆合规管理

### 3. 电子围栏域
- 围栏绘制管理（含3D立体围栏）
- 分区分类授权

### 4. 定位核心域
- 实时定位引擎
- 历史轨迹记录与查询（单人/多人/跨天）
- 室内外无缝切换
- 定位校准管理

### 5. 报警引擎域（核心）
- 人员聚集预警（三级：黄/橙/红）
- 越界报警（5s触发 + 30s升级）
- 静止报警
- 超员报警
- 缺员报警
- 滞留报警
- 一键SOS报警
- 报警静默管理

### 6. 报警处置域
- 报警闭环工作流（确认→分派→处理→关闭）
- 预警消警确认
- 聚集作业豁免管理

### 7. 可视化域
- 2D电子地图（多楼层）
- 3D可视化（透视穿墙）
- 人员信息叠加展示
- 实时统计看板
- 报警地图标注

### 8. 统计分析域
- 多维查询统计导出
- 报警趋势分析

### 9. 系统集成域
- 跨场景联动接口（双重预防/特殊作业/应急）
- 视频监控联动（摄像头跟随切换）

### 10. 园区级域
- 园区人员分布管理平台
- 园区应急数据支撑

### 11. 数据交换域
- 标准数据交换接口（API规范 + 安全传输）
- 增量数据同步（≤30s）

### 12. 数据存储域
- 数据保留策略（≥1年）
- 5张核心数据表（人员/区域/人员报警/区域报警/聚集预警）
