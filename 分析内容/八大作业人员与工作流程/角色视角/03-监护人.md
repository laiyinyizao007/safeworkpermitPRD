# 角色3: 监护人 - 界面设计方案

> **角色定位**: 全程在场的安全守门人
> **典型用户**: 持监护人证的安全员、班组安全骨干
> **主要终端**: 手机(常驻现场,不离手)

---

## 1. 角色画像

### 1.1 职责清单

- 全程在作业现场,不得脱岗
- 检查作业票与实际情况是否相符
- 录入气体检测数据(动火/受限空间)
- 实时监控作业人员状态
- 发现异常有权立即叫停作业
- 作业完成后参与验收签字

### 1.2 使用场景

| 场景 | 频率 | 终端 | 时长 |
|------|------|------|------|
| 现场核查(Verify阶段) | 每票1次 | 手机(现场) | 10-20分钟 |
| 气体检测录入 | 每2小时1次 | 手机(现场) | 2-3分钟 |
| 实时监护(常驻) | 全程 | 手机(现场) | 数小时 |
| 紧急叫停 | 极少 | 手机(现场) | 即时 |
| 验收签字 | 每票1次 | 手机(现场) | 3-5分钟 |

### 1.3 痛点与诉求

- 长时间在现场,手机耗电 → 界面轻量,低功耗模式
- 需要快速录入检测数据 → 数字键盘优先,支持IoT自动采集
- 紧急情况要一键叫停 → 叫停按钮常驻,无需翻页
- 被查岗时需要证明在场 → GPS轨迹自动记录

### 1.4 设计原则

- **常驻监护面板**: 作业期间始终显示关键状态
- **一键叫停**: 紧急按钮永远可见
- **自动记录**: GPS轨迹、在岗时间自动采集
- **定时提醒**: 气体检测到期自动弹窗

**交互逻辑**:
- 地理围栏校验: 不在作业点50m内 → 弹出警告,阻断核查
- 每项核查需拍照: 点击勾选 → 自动弹出相机
- 气体检测: 支持IoT蓝牙自动采集 或 手动录入数字键盘
- 数值超标自动报警: 可燃气体 ≥ 20%LEL → 红色警告,阻断"通过"按钮

- 气体检测倒计时: 到期自动弹窗提醒"请进行气体检测"
- GPS自动打卡: 每30分钟自动记录一次位置(无需手动操作)
- 监护日志: 自动生成,不可篡改

## 3. 通知与消息

| 事件 | 通知方式 | 内容 |
|------|---------|------|
| 被指定为监护人 | 推送+短信 | "您被指定为动火作业(1号储罐区)监护人" |
| 作业票审批通过 | 推送 | "动火作业票已审批通过,请前往现场核查" |
| 气体检测到期 | 弹窗+震动 | "距上次检测已过2小时,请立即检测" |
| 作业即将到期 | 推送+震动 | "作业将于1小时后到期" |
| GPS偏离告警 | 弹窗+震动 | "您已偏离作业区域,请立即返回" |
